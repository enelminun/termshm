# TERMShM
Shortcut Manager for Arch Linux

## Description
A dynamic Shortcut Manager for Arch Linux

# Installation

Copy the GitHub repository (or download it manually)
```bash
git clone https://github.com/enelminun/termshm.git
cd termshm
makepkg -si
```
